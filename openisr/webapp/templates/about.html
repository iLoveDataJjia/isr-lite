About this project

This project emerged after a frustation 😫 of having to pay for upscaling my images on the regular web applications such as Smart Upscaler, Bigjpg, Let's Enchance, ...
So as a data scientist, I decided to put my artificial intelligence skills to good use 🤪!
View on GitHub

For the AI enthusiasts 😍, OpenISR is mainly based on the EDSR and ESRGAN+ papers written in 2017 and 2020 respectively. As an open source project, the web application part is on purpose simple allowing easy codes understanding and modifications 👍. If you wish to contribute or modify the project according to your needs or checkout the papers, please feel free 😊. The codes and references are on the GitHub repository link just down below 👇.
